### ChatGPT를 활용한 레시피 검색



## 🔍 소개
준쉐프 Ver. 1의 문제점인 검색 속도를 개선하기 위해 진행한 프로젝트입니다.

---

## 🛠️ 기술 스택

**Language |** Java

**Framework |** Spring Boot, Spring MVC

**Build |** Gradle

**Database |** MySQL, H2

**ORM |** JPA(Hibernate)

**Infra |** AWS EC2

**Test |** JUnit5, Mockito, H2

**Tools |** Android Studio, IntelliJ

**Library |** ChatGPT
